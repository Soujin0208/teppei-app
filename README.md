## 🛠️ アップデートの しかた（こどもむけ）

### はじめるまえに

1. パパ/ママに `claude` を つかえるようにしてもらう
2. ターミナルで このフォルダ（`鉄平アプリ`）に いどうする

### ファイルを なおす（うえに つけたす かんじ）

1. ターミナルで `claude` と うつ
2. 「けいさんバトルの もんだいを もっと ふやしたい」みたいに おねがいする
3. Claude が なおしてくれる
4. できたら ブラウザで `gakushuu-portal.html` を ひらいて ためす

### みんなに しらせる（save & push）

```bash
git add .
git commit -m "○○を つけたした"
git push
```
