# 🎮 こどもたちへ — アプリを パワーアップさせよう！

ここを よむと、きみも このゲームを どんどん よくできるよ！
Claude（クロード）っていう すごい AI が、ぜんぶ てつだってくれる。

---

## 🚀 あそぶ だけなら

1. パソコンで `鉄平アプリ` の フォルダを ひらく
2. `gakushuu-portal.html` を ダブルクリック！
3. ブラウザで ゲームが はじまる ✨

---

## 🔧 ゲームを かえたい／つけたしたい ときは

### ステップ1：さいしんの バージョンに する

ターミナル（くろい がめんの アプリ）を ひらいて：

```bash
cd ~/鉄平アプリ
git pull
```

> 💡 `git pull` は「みんなが つくった さいしんを もらう」 まほう。

### ステップ2：Claude を よぶ

```bash
claude
```

Claude が あらわれる。

### ステップ3：おねがいする

こんな ふうに うつ：

- 「けいさんバトルの もんだいを もっと ふやして」
- 「てきを もっと かわいい どうぶつに かえて」
- 「かんじバトルに 小3の かんじを くわえて」
- 「せいかいしたときの 音を つけて」
- 「ファイターに パンダを ついかして」

Claude が かってに プログラムを なおしてくれる。

### ステップ4：ためす

`gakushuu-portal.html` を ブラウザで ひらいて、ちゃんと うごくか みる。

### ステップ5：みんなに シェアする

ちゃんと うごいたら、ターミナルで：

```bash
git add .
git commit -m "○○を つけたした"
git push
```

> 💡 `"○○を つけたした"` の ところは、じぶんが なにを やったか かくよ。たとえば `"けいさんの もんだいを 50こ ふやした"`

これで **GitHub** に のぼって、みんな（パパ・ママ・きょうだい）の パソコンにも つたわる！

## 📝 こんな かんじで Claude に はなそう

**OK な おねがい：**
- ✅ 「えいごバトルに どうぶつの もんだいを 10こ ふやしたい」
- ✅ 「せいかいの ときに きらきらが でるようにして」
- ✅ 「タイマーを 5びょう ながくして」
- ✅ 「ボスを もう1ぴき ふやして」

**うまく いかない ときの ヒント：**
- ❌ 「すごく おもしろくして」→ ぐあいてきに かいたほうが いい
- ✅ 「コンボが 10い じょうで とくべつな エフェクトが でるようにして」

---

## 🆘 こわれた とき

ゲームが こわれても、だいじょうぶ！

ターミナルで：
```bash
git log --oneline
```

むかしの バージョンの リスト が でる。もどしたい バージョンの 7もじ ID を メモ。

```bash
git checkout <7もじID> .
```

これで むかしに もどせる！ こわれたら パパに きくのも あり。
