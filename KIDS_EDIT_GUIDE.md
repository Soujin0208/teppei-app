# 🛠️ じぶんでゲームをつくろう！かいたつガイド

> このガイドは、**パソコンなしでもスマホだけで**ゲームをかいたり、あたらしいゲームをつくったりできるおしえかたです。
> プログラミングをはじめてやる人もだいじょうぶ！いっしょにやってみよう🎉

---

## 🌟 まずはこれをよんでね

このゲームのデータは **GitHub（ぎっとはぶ）** というばしょにあります。
GitHub は「みんなでつくったファイルをほぞん（保存）しておける場所(ばしょ)」です。
ウェブブラウザだけでファイルをひらいて、じぶんでなおすことができます。

まほうのURLはここ👇
```
https://github.com/Soujin0208/teppei-app
```

---

## 📝 ステップ１：ファイルをひらく

1. ブラウザで `https://github.com/Soujin0208/teppei-app` をひらく
2. ファイルのいちらん（一覧）がでてくるよ
3. なおしたいファイルの名前(なまえ)をクリック！
   - `calc-battle.html` → けいさんバトル
   - `math-adventure.html` → さんすうアドベンチャー
   - `kanji-battle.html` → かんじバトル
   - `english-battle.html` → えいごバトル
   - `gakushuu-portal.html` → トップページ

---

## ✏️ ステップ２：えんぴつアイコンをおす

ファイルをひらいたら、みぎうえのほうに **✏️（えんぴつのマーク）** があるよ。
それをおすと、ファイルをなおせるようになります！

```
みぎうえのあたりをさがしてみて →  [Raw]  [✏️]  [🗑️]
                                          ↑
                                     これをおす！
```

おしたら、もじがたくさんならんだ「へんしゅう(編集)がめん」になるよ。
はじめてみると「なんかいっぱいある…」とおもうかもしれないけど、
**なおしたいところだけさがして、すこしかえるだけでOK** です。

---

## 🔍 ステップ３：なおしたいところをさがす

へんしゅうがめんでは **Ctrl＋F（コントロールキーとFをいっしょにおす）** でけんさくできます。
スマホなら「…（みっつのてん）→ ページをけんさく」からできるよ。

---

## 🎮 かんたんカスタマイズ（こてい）れい

### ① キャラのなまえをかえてみよう

`calc-battle.html` や `math-adventure.html` をひらいて、こういうぶぶんをさがしてね：

```javascript
{e:'🐉',n:'ドラゴン'},
{e:'🦁',n:'ライオン'},
```

`n:'ドラゴン'` の **ドラゴン** のところをかえれば、じぶんのすきなまえにできるよ！

**れい：**
```javascript
{e:'🐉',n:'テッペイ'},   ← 「テッペイ」にかえた！
{e:'🦁',n:'ソウジン'},   ← 「ソウジン」にかえた！
```

---

### ② もんだいのむずかしさをかえてみよう

`math-adventure.html` をひらいて、`diff:2.0` をさがしてね：

```javascript
G.diff=1.0+gi*0.5;
```

このかずをかえると、はじまったときのむずかしさがかわります。
- `G.diff=1.0;` → いちばんやさしい！
- `G.diff=3.0;` → ちょっとむずかしい
- `G.diff=5.0;` → もうさいきょう！

---

### ③ バトルのもんだいすうをかえてみよう

`math-adventure.html` で `MAX_Q` をさがしてね：

```javascript
const MAX_Q=12;
```

ここのかずが「１バトルで何問(なんもん)でるか」です。
- `const MAX_Q=8;` → ８もんでおわり（みじかくてかんたん）
- `const MAX_Q=20;` → ２０もん（ちょうせんしたい人に！）

---

### ④ てきキャラの絵文字(えもじ)をかえてみよう

`math-adventure.html` をひらいて、`enemy:'🐛'` のようなところをさがしてね：

```javascript
{ name:'たし算（１ケタ）', type:'add1', enemy:'🐛', bossEnemy:'🐝', ...},
```

`🐛` のところをすきな絵文字(えもじ)にかえるだけ！
- `enemy:'🦖'` → きょうりゅうにかえた！
- `bossEnemy:'🐲'` → ボスはドラゴン！

---

## 💾 ステップ４：コミット（ほぞん）する

なおしおわったら、ページのいちばんしたにスクロールしてね。
**「Commit changes（こみっとちぇんじず）」** というボタンがあります。

```
┌─────────────────────────────────────┐
│ Commit changes                       │
│                                     │
│ ✏️ コミットメッセージをかこう        │
│ ┌──────────────────────────────┐   │
│ │ キャラのなまえをテッペイにかえた │   │
│ └──────────────────────────────┘   │
│                                     │
│  [● Commit directly to main]        │
│                                     │
│  [キャンセル]  [Commit changes ✅]  │
└─────────────────────────────────────┘
```

1. コミットメッセージ（なにをかえたか）を日本語(にほんご)でかいてOK！
   - れい：「キャラのなまえをかえた」
   - れい：「もんだいすうを20にした」
2. **「Commit directly to main」** がえらばれていることをかくにんしてね
3. みどりの **「Commit changes」** ボタンをおす！

これで **ほぞんかんりょう！** 🎉
すこしまってからゲームのURLをひらくと、かわっているはず！

## 🆕 あたらしいゲームをつくりたいとき

### ① まず元(もと)のファイルをコピーする

1. `https://github.com/Soujin0208/teppei-app` をひらく
2. コピーしたいファイル（たとえば `math-adventure.html`）をひらく
3. みぎうえの **✏️（えんぴつ）** をおす
4. **Ctrl＋A（ぜんぶえらぶ）** → **Ctrl＋C（コピー）**
5. ページをもどって、みぎうえの **「＋（たす）」** ボタン → 「Create new file（あたらしいファイルをつくる）」をおす
6. ファイルめいをきめる（たとえば `science-battle.html`）
7. **Ctrl＋V（はりつける）** でコピーしたないようをはりつける

### ② ゲームのなかみをかえる

コピーしたファイルをかいたつして、じぶんだけのゲームにしよう！

かえるとおもしろいところ：

| かえるばしょ | れい |
|------------|------|
| `<title>` タグのなか | `<title>りかバトル！</title>` |
| ゲームタイトル | `さんすうアドベンチャー` → `りかバトル！` |
| キャラのなまえ（`CHARS`） | すきなどうぶつや人のなまえ |
| もんだいのタイプ（`GRADES`） | じぶんでかんがえたもんだいに |

### ③ ポータルにつなぐ

`gakushuu-portal.html` に あたらしいゲームのカードをついかすれば、
トップページからあそべるようになります。
