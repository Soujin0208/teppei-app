## 1. 検討したアプリのアイデア

### 発端
息子たち（四男：5歳、三男：小4）に**足し算・引き算を楽しくやらせたい**という要望から開発スタート。

### 検討・採用したアイデア

| アイデア | 採用 | 概要 |
|----------|------|------|
| シンプルなフラッシュカード型 | ❌ | 単調すぎる |
| タイムアタック型 | ❌ | プレッシャーが強すぎる |
| **スマブラ風バトル系** | ✅ | 問題に正解すると敵を攻撃。敵を倒しながら進む |
| 漢字バトル | ✅ | 計算バトルと同じ仕組みで漢字読み・意味を学ぶ |
| 英語バトル | ✅ | 英単語の日英・英日変換。幼稚園児にも対応 |
| 統合ポータル | ✅ | 3ゲームをまとめて選べるトップページ |

## 2. 決まった仕様・機能

### 2-1. 共通仕様（3ゲーム共通）

#### ゲーム構造
- **ストーリーモード**：雑魚敵10体 → ボス1体（全11ステージ）
- **2先取制**：2ラウンド先取で勝利
- **セーブ機能**：`localStorage` でレベル・経験値・実績を保存（ゲームごとに独立したキー）

#### キャラクター
- プレイヤーキャラ **16体** から選択（4列×4行グリッド）
- 各キャラに固有スキルあり（スキルゲージが満タンで発動）

| スキル種別 | 内容 |
|-----------|------|
| burst | 3問分のダメージ一括 |
| combo | コンボ×3スタート |
| auto | 次の問題を自動正解 |
| percent | 敵HP30%削る |
| heal | HP30回復 |
| crit | 次の攻撃×4 |
| poison | 毎問5ダメージ |
| atkup | ATK×2（10秒） |
| shield | 次の敵攻撃を0にする |
| zap | まとめて50ダメージ |
| prism | スコア×1.5（5問） |
| lucky | HPとスキルゲージ全回復 |
| nobreak | コンボ切れなし（5問） |
| triple | 次のダメージ×3 |
| half | 敵HPを半分にする |
| miracle | HPフル回復＋ATK永続+10 |

#### 演出
- **ボス登場前**：雑魚キャラが画面四方からわらわら出現→中央に向かって突進する演出（28体、約2.4秒）
- タイマーボーナス：早く答えるほど追加ダメージ
- コンボシステム：連続正解でダメージ倍率アップ
- KO演出・ステージクリア演出

#### 解説システム
- 正解・不正解どちらでも解説ボックスを表示
- 「わかった！つぎへ」ボタンを押さないと次の問題に進めない
- 解説タイトルを「⭕ せいかい」「❌ まちがい」「⏰ じかんきれ」で出し分け

#### 弱点追跡
- 間違えた問題を記録し、30%の確率で優先出題
- リザルト画面で弱点ワード/漢字トップ8を表示

### 2-2. けいさんバトル（`calc-battle.html`）

**対象**：四男（5歳）〜三男（小4）

| 項目 | 内容 |
|------|------|
| 問題形式 | 四則演算（+/−/×/÷）の穴埋め |
| 難易度 | diff 1〜6（1桁加減→2桁→繰り上がり→乗除） |
| タイマー | 18秒 |

---

### 2-3. かんじバトル（`kanji-battle.html`）

**対象**：小学1〜2年生相当

| 項目 | 内容 |
|------|------|
| 問題形式 | 漢字のよみかた（4択） |
| 漢字データ | 小学1〜2年生漢字 約78文字 |
| 難易度区分 | diff 1〜6（数字・自然→動植物→からだ→人・学校→場所→総合） |
| タイマー | 18秒 |

---

### 2-4. えいごバトル（`english-battle.html`）

**対象**：幼稚園児〜小学低学年

| 項目 | 内容 |
|------|------|
| 問題形式 | 英→日（e2j）、日→英（j2e）の4択 |
| 単語数 | 約100語（色・数・動物・食べ物・乗り物・天気・学校・スポーツ等） |
| 難易度 | diff 1〜4 |
| 幼稚園対応 | 選択肢に絵文字＋ひらがな表示、タイマー22秒に延長 |
| diff1問題 | 英語→日本語のみ（絵文字ヒント大表示56px） |

### 2-5. 統合ポータル（`gakushuu-portal.html`）

- 3ゲームのトップページ
- 各ゲームの現在レベルを`localStorage`から読み取り表示
- 各ゲームに「もどる」ボタンでポータルに戻れる

## 3. 使う技術（言語・フレームワーク）

### 言語
- **HTML / CSS / JavaScript**（単一ファイル完結）
- フレームワーク・ビルドツールなし（依存ゼロ）

### フォント
- `Boogaloo`（英数字・ゲームタイトル）— Google Fonts CDN
- `Noto Sans JP`（日本語）— Google Fonts CDN

### データ永続化
- `localStorage`（ブラウザ内、サーバー不要）
- キー名：`calcBattle_v1` / `kanjiBattle_v1` / `engBattle_v1`

### 動作環境
- iPhone Safari / Android Chrome を主ターゲット
- `viewport width=device-width, maximum-scale=1.0`
- `-webkit-tap-highlight-color: transparent` でタップUI最適化
- `100svh` 対応（スマホのアドレスバー考慮）

### ファイル構成
```
gakushuu-portal.html   ← エントリーポイント
calc-battle.html
kanji-battle.html
english-battle.html
```
※ 全ファイルを**同一フォルダ**に置いて使用
